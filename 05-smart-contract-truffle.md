# Truffle Framework
Build - Run - Test - Deploy

# Setup truffle
```shell
npm install -g truffle
```

# Setup testrpc
```shell
# Install
npm install -g ethereumjs-testrpc

# Run
testrpc
```

# Setup project
```shell
truffle init
```

# Build
```shell
truffle compile
```

# Console
```shell
# To getting in
truffle console

# To getting out
.exit
```
